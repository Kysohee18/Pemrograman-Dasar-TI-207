Repositori Mata Kuliah Pemrograman Dasar C++ (TI203)
Repositori ini adalah sumber daya pendukung untuk mahasiswa semester 2 Program Studi Teknik Informatika (TI) Universitas Muhammadiyah Yogyakarta (UMY) yang menempuh mata kuliah TI203 Pemrograman Dasar dan TI203 P Praktikum Pemrograman Dasar . 
Tujuannya adalah untuk menyediakan materi, contoh kode, dan latihan praktikum guna membantu pemahaman konsep pemrograman menggunakan bahasa C++.
